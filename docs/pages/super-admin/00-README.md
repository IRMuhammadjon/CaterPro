# Super Admin (Platforma Egasi) Sahifalari

> CaterPro platformasi egasi (Siz) uchun barcha sahifalar
> Barcha tashkilotlarni boshqarish, monitoring va analitika

## Sahifalar Ro'yxati

| # | Sahifa | Route | Tavsif |
|---|--------|-------|--------|
| 01 | Login | `/super-admin/login` | Super admin kirish |
| 02 | Dashboard | `/super-admin/dashboard` | Bosh panel |
| 03 | Organizations | `/super-admin/organizations` | Tashkilotlar ro'yxati |
| 04 | Organization Detail | `/super-admin/organizations/:id` | Tashkilot tafsiloti |
| 05 | Create Organization | `/super-admin/organizations/create` | Yangi tashkilot |
| 06 | Subscriptions | `/super-admin/subscriptions` | Obunalar boshqaruvi |
| 07 | Payments | `/super-admin/payments` | To'lovlar |
| 08 | Analytics | `/super-admin/analytics` | Analitika |
| 09 | Reports | `/super-admin/reports` | Hisobotlar |
| 10 | Settings | `/super-admin/settings` | Sozlamalar |
| 11 | Support | `/super-admin/support` | Support ticketlar |
